# Extension Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

You can create simple icons using any image editor or use an icon generator.

**Temporary workaround:** Create simple PNG files with a colored background and text "RD" (Remote Desktop).

**Or use an online tool:**
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
